# NetFare InDrive

تطبيق أندرويد بيقرأ شاشة تطبيق InDrive (السواق) عن طريق Accessibility Service،
ويحسب صافي السعر بالكيلومتر بعد خصم نسبة عمولة الشركة، وبيعرضها في Badge عايم
فوق أي تطبيق.

## خطوات التشغيل

### 1. رفع المشروع على GitHub
- اعمل repo جديد على GitHub (لو مش عامل قبل كدة)
- ارفع كل الملفات دي جواه (فك الضغط أول حاجة)
- تأكد إن اسم الفرع الرئيسي هو `main` (أو غيّر اسم الفرع في `build.yml`)

### 2. تشغيل الـ Build
- بمجرد الرفع، الـ GitHub Actions هيشتغل تلقائي (تقدر تتابعه من تبويب Actions)
- أو شغله يدوي من تبويب Actions -> Build APK -> Run workflow

### 3. تنزيل الـ APK
- بعد ما الـ workflow يخلص (ياخد 2-4 دقايق تقريباً)
- روح لصفحة الـ run -> هتلاقي في الأسفل "Artifacts" -> نزل `NetFare-debug-apk`
- ده ملف zip فيه الـ APK جواه (`app-debug.apk`)

### 4. تثبيت وتشغيل على الموبايل
- فعّل "تثبيت من مصادر غير معروفة" في إعدادات الأندروید
- ثبّت الـ APK
- افتح التطبيق:
  - سيب نسبة العمولة زي ما هي أو غيّرها واضغط حفظ
  - اضغط "تفعيل خدمة قراءة الشاشة" وفعّل NetFare InDrive من قايمة Accessibility
  - اضغط "تفعيل صلاحية الظهور فوق التطبيقات" ووافق
- افتح تطبيق InDrive بتاع السواق، والـ Badge هيظهر فوق ويحدث صافي السعر/كم أول
  ما يلاقي رقم سعر ومسافة على الشاشة

## ملاحظة مهمة

الكود بيدور على نص فيه رقم متبوع بـ "EGP"/"جنيه"/"ج.م" ورقم متبوع بـ "km"/"كم"
في شاشة InDrive. لو التطبيق مش لاقي القيم صح، ابعتلي لقطة شاشة أو نص فعلي
لشكل السعر والمسافة زي ما بتظهر في InDrive، وأنا أظبط الـ regex بتاع الكود
(`FareAccessibilityService.java`) على شكلها بالظبط.
